# 🥗 Food Remedy

**Food Remedy** is a mobile-first platform that helps Australian consumers make **smarter, healthier food choices**.  

📱 Users can scan supermarket products to see:  
- Nutritional information  
- Allergen warnings  
- Dietary suitability  

Everything is delivered through a **personalised mobile experience**.  

This project uses:  
- **Expo Go (React Native)** → mobile app development  
- **Google Firebase** (Firestore + Authentication) → backend

## 📦 Initialise Repository

### Step 1: Clone the Repository
```bash
git clone https://github.com/Gopher-Industries/Food-Remedy_API.git
```

### Step 2: Create a New Branch
```bash
cd Food-Remedy_API
git checkout -b your-branch-name
```

### 🔖 Ticket Name 
Naming Rule: Use ticket ID + title.  
Example: For the following ticket `FE007-Integrate-and-Test-Week-3-Changes`  
```bash
git checkout -b FE007 - Integrate and Test Week 3 Changes
```


### Notes: 
⚠️ Never push changes directly to main.  
When ready, push your branch and create a pull request.
