🚀 HOW TO USE RIGHT NOW

1. Start your app:
   $ npm start

2. Go to Scan tab in your app

3. Point camera at barcode/QR code

4. App validates against Firebase PRODUCTS collection

5. If found → Auto-navigates to product details
   If not found → Shows error, can retry or search
